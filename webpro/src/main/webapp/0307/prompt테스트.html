버튼을 클립하면 두개의 수를 입력한다 (prompt 이용)
  prompt함수의 리턴겨로가 값은 String이다.
  입력받은 두 수를 더해서 출력한다 - div요소 더하기 위해서 정수로 변환이 필요 - praseInt()
  
  요소검색 - es6 - 단수로 검색
  document.querySelector("#aa")  -> 아이디 검색
  document.querySelector(".bb")  -> 클래스 검색
  document.querySelector("div")  -> 태그 검색
   ※ 클래스와 태그가 복수개가 있어도 가장 처음에 찾아낸 요소의 값만을 가져온다.
