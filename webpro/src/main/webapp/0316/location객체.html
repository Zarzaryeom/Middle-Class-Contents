location 객체
    url정보를 얻는 객체
    host : hostname + port
    hostname : 서버 이름
    port : 포트 번호(80일 경우 생략)
    href : 전체 URL로 페이지 설정
    
    버튼 클릭하면 'location.href = "페이지이름"' 을 이용하여 이동한다.
     -> 서버내에서 처리 후 서버페이지로 이동
     -> 서버내에서 실행한 결과값을 가지는 페이지로 이동 가능
     -> 예 : 로그인 실행한 jsp페이지거나 게시판 글쓰기를 수행한 servlet파일 이거나 등등
    
    
    'a href="페이지이름"'은 서버 프로그램 수행 없이 단순히 클라이언트에 보여지는 페이지 이동
